Anthropic이 1월 12일 Cowork를 발표했다. Claude Desktop에 탑재된 이 기능은 코딩 지식 없이도 AI가 로컬 파일을 읽고, 수정하고, 생성하게 해준다. "AI 동료에게 메모 남기듯" 업무를 맡기는 시대가 열렸다.

## Cowork란 무엇인가

Cowork는 Claude Desktop 앱에 내장된 AI 에이전트다. 사용자가 지정한 폴더 내에서 Claude가 자율적으로 파일 작업을 수행한다.

**핵심 기능:**
- 파일 읽기, 수정, 생성
- 다중 작업 동시 처리
- 자체 검증 및 오류 시 질문

기존 Claude Code가 개발자용 터미널 도구였다면, Cowork는 일반 사용자를 위한 데스크톱 버전이다.

## 활용 예시

Anthropic이 제시한 대표 사용 사례:

| 작업 | 설명 |
|------|------|
| 다운로드 폴더 정리 | 파일 분류 및 지능형 이름 변경 |
| 영수증 분석 | 스크린샷에서 비용 추출해 스프레드시트 생성 |
| 문서 초안 작성 | 여러 노트에서 정보 취합해 보고서 생성 |
| 데이터 정리 | CSV 파일 정제 및 구조화 |

핵심은 **자율 실행**이다. 단순히 텍스트를 생성하는 게 아니라, 계획을 세우고 → 단계별 실행 → 자체 검증 → 막히면 질문하는 흐름으로 작동한다.

## 10일 만에 탄생한 배경

Cowork의 개발 기간은 단 10일이다.

2024년 말 출시된 Claude Code는 개발자들 사이에서 큰 인기를 끌었다. 그런데 Anthropic 엔지니어 Boris Cherny에 따르면, 흥미로운 현상이 발견됐다. 사용자들이 **코딩 도구로 비코딩 작업을 처리**하고 있었던 것.

"사용자들이 Claude Code를 파일 정리, 이메일 초안, 스프레드시트 작업에 활용하는 걸 봤습니다. 이 행동을 막는 대신, 전용 제품을 만들기로 했죠."

지난 주 반 만에 Cowork가 완성됐다.

## 사용 방법

1. Claude Desktop 앱 설치 (macOS)
2. 작업할 폴더 지정 (샌드박스)
3. 채팅으로 지시 전달
4. Claude가 자율적으로 실행

여러 작업을 큐에 넣으면 Claude가 동시에 처리한다. Anthropic은 이를 "동료에게 메모를 남기는 느낌"이라고 표현했다.

## 이용 조건과 제한

**현재 이용 가능:**
- Claude Max 구독자 전용
- macOS만 지원 (Windows 예정)
- Research Preview 단계

**대기자 명단:**
Pro 및 기타 플랜 사용자는 대기자 등록 가능

## 주의사항

Anthropic은 몇 가지 위험 요소를 공식 경고했다.

**보안 위험:**
- 프롬프트 인젝션 공격 가능성
- 파일 삭제 권한 보유
- 모호한 지시 시 예기치 않은 동작

**권장 사항:**
- 지시는 최대한 명확하게
- 중요 파일은 백업 후 작업
- 민감 정보가 있는 폴더는 지정하지 말 것

## Claude Code vs Cowork

| 항목 | Claude Code | Cowork |
|------|-------------|--------|
| 대상 | 개발자 | 일반 사용자 |
| 인터페이스 | 터미널 | GUI 채팅 |
| 작업 범위 | 코드 작성/수정 | 파일 관리/문서 작업 |
| 기술 요구 | 높음 | 낮음 |

## FAQ

**Q: 무료로 사용할 수 있나요?**

A: 현재는 Claude Max 구독자만 이용 가능합니다. 다른 플랜은 대기자 명단에 등록할 수 있습니다.

**Q: Windows에서도 쓸 수 있나요?**

A: 아직 macOS만 지원합니다. Windows 버전은 추후 출시 예정입니다.

**Q: 클라우드에 파일이 업로드되나요?**

A: Cowork는 로컬 폴더에서 작동합니다. 다만 AI 처리를 위해 일부 데이터가 Anthropic 서버로 전송될 수 있으므로, 민감 파일 작업 시 주의가 필요합니다.

## 결론

Cowork는 AI 에이전트의 대중화를 앞당기는 제품이다. 코딩을 모르는 직장인도 AI에게 반복 업무를 위임할 수 있게 됐다.

물론 아직은 Research Preview다. 파일 삭제 권한 같은 위험 요소도 있다. 하지만 방향성은 명확하다. AI가 단순한 챗봇을 넘어 **실제로 일을 처리하는 동료**로 진화하고 있다.

"Claude Code 사용자들이 원하는 걸 관찰하고, 10일 만에 새 제품을 만들었다." Anthropic의 이 민첩함이 AI 업계의 속도를 보여준다.
