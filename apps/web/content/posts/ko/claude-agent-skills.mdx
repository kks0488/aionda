범용 AI에게 전문적인 작업을 시킬 때마다 긴 프롬프트를 매번 입력하는 것이 지치지 않나요? (문제)
Anthropic은 2025년 10월, **'Agent Skills'**라는 새로운 개념을 도입했습니다. 이는 특정 작업에 필요한 지침, 스크립트, 리소스를 하나의 폴더(Skill)로 묶어 AI에게 장착시키는 기능입니다. (해결책)
이제 개발자는 도메인 전문 지식을 '스킬'로 패키징하여 조직 전체에 배포할 수 있습니다. (근거)

## Agent Skills란 무엇인가?

### 1. Anthropic-managed Skills (기본 스킬)
Anthropic이 미리 만들어 둔 고품질 스킬들입니다.
- **Office Skills**: 파워포인트(PPTX), 엑셀(XLSX), 워드(DOCX) 파일을 자유자재로 다루는 스킬.
- **PDF Analyst**: 복잡한 표와 차트가 포함된 PDF 문서를 정밀 분석하는 스킬.

### 2. Custom Skills (사용자 정의 스킬)
기업은 자신들만의 워크플로우를 스킬로 만들 수 있습니다.
- 예: "우리 회사의 코드 스타일 가이드에 맞춰 리팩토링하는 스킬"
- 예: "월간 마케팅 보고서 형식을 준수하여 데이터를 정리하는 스킬"

## 작동 방식
스킬은 단순한 프롬프트 모음이 아닙니다. **코드 실행(Code Execution) 도구**와 결합되어, 실제 Python 스크립트를 통해 파일을 조작하고 데이터를 처리합니다. 즉, Claude가 '말'만 하는 것이 아니라 '손'을 쓰게 되는 것입니다.

## 도입 효과
이 기능의 도입으로 Claude는 단순한 챗봇을 넘어, **특정 직무를 수행하는 디지털 사원**처럼 활용될 수 있습니다. 온보딩 비용 없이 즉시 투입 가능한 전문가를 고용하는 것과 같습니다.
