"Sora 2로 영상을 만들어보려는데 생성이 안 되나요?" (문제)
2026년 1월 10일을 기점으로 **OpenAI가 Sora 2의 무료 티어(Free Tier) 영상 생성 기능을 공식적으로 중단**했습니다. 이제 영상 생성은 월 20달러 이상의 **Plus 또는 Pro 구독자만의 특권**이 되었습니다. (해결책)
갑작스러운 정책 변경의 배경에는 폭발적인 GPU 비용 증가와 고품질 서비스 유지를 위한 불가피한 선택이 있었습니다. 이번 정책 변경의 세부 내용과 사용자별 대응 전략을 정리했습니다. (근거)

## 💸 무엇이 바뀌었나? (Policy Update)

### 1. 무료 사용자(Free Tier) 접근 차단
- **기존**: 제한된 횟수 내에서 Sora 2 영상 생성 가능
- **변경**: 영상 생성 **불가능**. (이미지 생성도 일부 제한될 수 있음)
- **이유**: Sora 2의 렌더링 비용이 텍스트 모델(GPT-5) 대비 수십 배 높기 때문에, 무료 트래픽을 감당하기 어려워진 것으로 분석됩니다.

### 2. Plus / Pro 구독자 혜택 강화
유료 사용자에게는 더 빠르고 안정적인 생성 대기열(Priority Queue)이 제공됩니다.
- **Plus ($20/월)**: 표준 속도, 일일 생성 한도 적용
- **Pro ($200/월)**: 고속 모드, 상업적 이용 권한, 더 높은 생성 한도

## 왜 지금인가? (Background)

Sora 2는 출시 직후 헐리우드 수준의 영상 품질로 충격을 주었지만, 그만큼 **'전기 먹는 하마'**라는 비판도 받았습니다. 
최근 사용자 수가 급증하면서 서버 부하가 임계치에 도달했고, OpenAI는 **"서비스 품질 유지"**를 명분으로 무료 이용 종료 버튼을 눌렀습니다. 이는 DALL-E 3 초기 시절의 제한 조치와 유사하지만, 훨씬 더 급진적입니다.

## 👉 사용자를 위한 대응 가이드

### 1. 일반 사용자 (Casual User)
가끔 재미로 영상을 만들던 사용자라면, 월 20달러 결제가 부담스러울 수 있습니다.
- **대안**: **Runway Gen-3 Alpha**나 **Luma Dream Machine**의 무료 티어를 활용해 보세요. 아직 제한적이지만 무료 사용이 가능합니다.

### 2. 크리에이터 및 개발자 (Pro/Dev)
Sora 2를 업무에 활용한다면 유료 구독은 필수입니다.
- **API 사용자**: API 정책도 변경되었습니다. API를 통한 호출은 별도 과금되므로, 기존 앱 서비스에 Sora 2를 연동했다면 **비용 폭탄**에 주의해야 합니다. 반드시 `usage limit`을 설정하세요.

## FAQ: Sora 2 유료화 관련 질문

### Q1. 기존에 무료로 만든 영상은 삭제되나요?
**A.** 아니요, 기존에 생성한 영상은 보존됩니다. 단, '추가 생성'이나 '리믹스' 기능은 유료 구독 전환 전까지 잠깁니다.

### Q2. GPT-5 텍스트 채팅도 유료화되나요?
**A.** 현재로서는 영상(Sora 2) 생성 기능에만 국한된 조치입니다. 텍스트 모델인 GPT-5.1 / 5.2 등은 여전히 무료 티어에서도 (제한적으로) 사용 가능합니다.

### Q3. 학생 할인은 없나요?
**A.** 현재 공식적인 학생 할인은 없으며, ChatGPT Edu 플랜을 도입한 학교 계정을 통해서만 접근이 가능할 수 있습니다.
