## 문제 인식
### 1) “구닥다리” 토픽이 자주 뜨는 이유(추정)
- 커뮤니티/요약글 중에 “옛 모델 비교/회고”형 제목이 많으면, 토픽 추출 단계에서 그 패턴이 **트렌드로 과대 대표**될 수 있다.
- 출처 선택이 `raw`(커뮤니티) 비중이 높으면, 최신 공식 발표/문서/뉴스보다 커뮤니티의 회고/밈/리뷰가 더 자주 잡힌다.

### 2) “네이버에는 있는데 본진에는 없다”
- 현 상태 기준으로, 메인 사이트(`aionda.blog`)는 GitHub `main`에 들어간 MDX(콘텐츠)로 배포된다.
- 네이버 블로그 업로드는 별도의 채널이며, **자동 동기화가 보장되지 않는다**(수동 업로드/별도 자동화 여부에 따라 달라짐).
